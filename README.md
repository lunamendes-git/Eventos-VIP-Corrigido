# Eventos-VIP-Corrigido
Eventos VIP é um sistema de gestão de eventos desenvolvido para organizar convidados, mesas, garçons, pedidos e pagamentos. A aplicação permite configurar temas personalizados para cada evento, gerenciar cardápios exclusivos, registrar pedidos com descontos automáticos para convidados VIP e gerar relatórios completos de faturamento.
